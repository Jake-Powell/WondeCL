# WondeCL <img src="man/figures/logo.png" align="right" height="139" alt="" />

<!-- badges: start -->
<!-- badges: end -->

The goal of WondeCL is to make is easy to use the Wonde API to create
class lists to be used for sending surveys to schools with cohort
studies.

The package also includes functionality including:

- manually editing class lists
- Manually adding schools to the class list created using Wonde data
- methods to withdrawn students who are withdrawn from the study.

## Installation

You can install the development version of WondeCL like so:

``` r
install.packages('devtools')
devtools::install_github('Jake-Powell/WondeCL')
```
